
  # NewsAgg

## Installation
1. Fork and clone repository
2. Navigate to the `server` directory and create a .env file
3. In the `.env` file, add your NewsAPI key in the format `API_KEY=your_newsapi_key`
4. Run `npm install`
5. Run `npm start` or `node server.js` to start the server
6. Navigate to the `client` directory and run `npm install`
7. Run `npm run dev` to start the client
